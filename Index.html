SEMANA VI - Position y Pseudoelementos

Atributos de posiciones: position

Hay cinco posibles valores: static|absolute|fixed|relative|sticky|initial|inherit

Valor Descripcion
static Valor por defecto. Los elementos se renderizan en el mismo orden en el que aparecen en el código
absolute El elemento se posiciona relativo a su primer elemento ascendente (ancestro, emparentado) que no tiene posicion estática
fixed El elemento se posiciona relativo a la ventana del navegador
relative The element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position
sticky The element is positioned based on the user's scroll position
A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Pseudoelementos

element: nth-child(i) n= teniendo un numero de hijos desde 0 a n, i es el elemento que queremos encontrar.
Sería como el i-abo elemento. Ejemplo, si tenemos 10 elementos e i==3 estamos seleccionando el tercer elemento hijo que es del tipo "element". También se pueden seleccionar solo los elementos pares o incluso cada N elementos (4n es cada cuatro elementos). Esto se puede combinar como en la tabla de arriba para seleccionar, por ejemplo, los elementos pares hijos de los elementos impares de los hijos de un elemento para hacer, por ejemplo, una seleccion tipo tablero de ajedrez tal y como aparece en la tabla de arriba.
Se puede ver como seleccionar elementos de cierto tipo/clase que cumplan la condicion del pseudoelemento

la etiqueta "colgroup" se usa para dar un estilo a toda una columna. Se aplica de forma que la etiqueta "col" que se pone dentro de la etiqueta se aplica a la primera columna de la tabla. Col(1) se aplica a x_1_1...x_i_1, es decir, col(j) -> x_1_j ... x_i_j. Se puede usar span para extender su rango d eaplicacion a mas de una columna contigua

Etiquetas multimedia

Las etiquetas multimedia se usan para introducir, valga la redundancia, archivos multimedia de audio, video, etc. Al igual que el resto de etiquetas tambien pueden llevar atributos, algunos exigen valores pero tambien pueden ser booleanos y que solo sea necesario poner el nombre del atributo, por ejemplo, "autoplay", "loop" o "controls". Las etiquetas multimedia requieren de una fuente o source (atributo src o con otra etiqueta dentro de la etiqueta correspondiente que sea "source").
